# Playwright Learning Project

This is a learning repository for Playwright, organized by chapters.

## Project Structure
- Chapters are in folders like `Chaper_1_basic/` (note: "Chaper" is a typo for "Chapter")
- Lesson files are numbered, e.g., `01_basic_js`

## Running Code
Use Node.js to run JavaScript files:
```
node Chaper_1_basic/01_basic_js
```

## Conventions
- Follow Playwright best practices when adding tests
- Use descriptive test names
- Organize tests by page or feature

## Pitfalls
- Folder names may have typos (e.g., "Chaper" instead of "Chapter")
- Ensure file extensions are correct (.js for JavaScript)

## Key Files
- Lesson files exemplify basic concepts progressing to advanced Playwright usage

For Playwright documentation, see [official docs](https://playwright.dev/).
